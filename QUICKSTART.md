# Quick Start Guide

Get up and running in 5 minutes!

## 1. Run Setup Script

**macOS/Linux:**
```bash
./setup.sh
```

**Windows:**
```bash
setup.bat
```

## 2. Add Your API Key

Edit `.env` and add your DeepSeek API key:
```env
DEEPSEEK_API_KEY=your_key_here
```

Get your API key from: https://platform.deepseek.com/

## 3. Run the Application

**macOS/Linux:**
```bash
source venv/bin/activate
python3 main.py
```

**Windows:**
```bash
venv\Scripts\activate
python main.py
```

## 4. Create Your First Mind Map

1. Click `File → New Mind Map` (or press `Ctrl+N` / `Cmd+N`)
2. Enter some text (e.g., "Python programming basics")
3. Click "Create"

That's it! Your mind map is created and saved automatically.

## Need Help?

- See [README.md](README.md) for detailed documentation
- Check the Troubleshooting section for common issues
- Verify your Python version: `python3 --version` (should be 3.8+)

## What's Next?

- Edit nodes: `Edit → Edit Current Mind Map`
- Export images: `File → Save Image`
- Explore the saved mind maps in the left panel

Happy mind mapping! 🗺️
